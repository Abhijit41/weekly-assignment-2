
# Celebal Technologies Data Engineer Internship - SQL Assignment

## Overview
This project was completed as part of the **Celebal Technologies Data Engineer Internship**. It demonstrates SQL-based data analysis on an E-commerce Sales dataset using **Python, SQLite, Pandas, and Google Colab**.

## Dataset
- Dataset: E-commerce Sales Data
- File: `data.csv`
- Total Records: **541,909**
- Total Columns: **8**

### Dataset Columns
- InvoiceNo
- StockCode
- Description
- Quantity
- InvoiceDate
- UnitPrice
- CustomerID
- Country

## Technologies Used
- Python
- Pandas
- SQLite
- SQL
- Google Colab

## Project Workflow
1. Loaded the CSV dataset using Pandas.
2. Created an SQLite database.
3. Imported the dataset into the `sales` table.
4. Executed SQL queries for data analysis.
5. Generated insights from sales data.

## SQL Queries Performed
- Total number of orders
- Revenue generated by each invoice
- Duplicate invoice detection
- Highest-priced products
- Average order value by country
- Total quantity sold by product
- Monthly revenue analysis
- Country-wise order analysis
